# Required Project Context

Before performing implementation work, read [COMPATIBILITY-GOAL.md](COMPATIBILITY-GOAL.md).

The repository's executable source, tests, fixtures, corpora, and generated
validation evidence define current project state.

Do not rely on commit messages, historical reports, documentation claims,
comments, or prior model summaries without independently confirming them.

Do not approximate unsupported compatibility behavior. Implement it from
authoritative evidence or report the concrete unresolved requirement.
